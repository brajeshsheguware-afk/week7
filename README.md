##  PROJECT OVERVIEW

The **COVID-19 Data Analysis & Interactive Dashboard System** is a data analytics project that examines the global impact of the COVID-19 pandemic using a real-world dataset obtained from Kaggle.

The project focuses on analyzing key pandemic indicators such as:

* Confirmed Cases
* Death Cases
* Recovered Cases
* Active Cases

Using **Exploratory Data Analysis (EDA)** techniques, the system identifies patterns, trends, and relationships within COVID-19 data across multiple countries.

An **interactive dashboard** built with **Plotly Dash** enables users to explore country-wise COVID-19 statistics dynamically through visualizations and real-time interaction.

---

##  PROBLEM STATEMENT

During the COVID-19 pandemic, enormous amounts of health-related data were generated daily across the world.

Organizations such as:

* World Health Organization (WHO)
* Johns Hopkins University
* Government Health Agencies

continuously published pandemic statistics including confirmed cases, recoveries, and deaths.

Manually analyzing such large-scale datasets is difficult and inefficient. Therefore, an automated analytical system is required to:

* Process pandemic data efficiently
* Discover meaningful trends
* Generate visual insights
* Support data-driven decision making
